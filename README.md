## Inicializar y Ejecutar el Servidor Posterior (Backend via WSL/Linux)

Por favor, utiliza tu terminal o WSL para lo siguiente en estricto orden:

1. Ubícate dentro de la carpeta local de la base de datos y la API:
   ```bash
   cd "Backend/Fake Store"
   ```
2. Crea una burbuja de aislamiento de dependencias de Python (Virtual Entorno):
   ```bash
   python3 -m venv venv
   ```
3. Activa tu nuevo entorno virtual para poder ejecutar comandos limpios:
   ```bash
   source venv/bin/activate
   ```
4. Agrega los requisitos pre-fabricados por el profesor en él:
   ```bash
   pip install -r requirements.txt
   ```
5. Enciende el motor oficial del Backend:
   ```bash
   fastapi dev
   ```
